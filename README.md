# walmart-smack

### Discipline project that simulates Walmart's system concurrently and efficiently that works with SMACK technologies, which are:
* [Apache Spark](https://spark.apache.org/)
* [Apache Mesos](http://mesos.apache.org/)
* [Akka](https://akka.io/)
* [Apache Cassandra](http://cassandra.apache.org/)
* [Apache Kafka](https://kafka.apache.org/)
